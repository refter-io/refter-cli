## Setup

### Requirements

* Python 3.10+

### Installation

Install it directly into an activated virtual environment:

```text
$ pip install refter-cli
```

## Usage

Check out the [refter](https://refter.io/docs) documentation to learn more about how to use the `refter-cli` client.
